Ambiente de Validação & Homologação

Validação visual e funcional do tema DSGovBR para Django 5.2 LTS e 6.0.

Esta aplicação de referência (sandbox) foi projetada para testar o comportamento visual e estrutural do tema baseado no Design System do Governo Federal (dsgovbr). Ela cobre as superfícies críticas do Django que costumam apresentar quebras em temas customizados: o painel administrativo (django.contrib.admin), fluxos nativos de autenticação (django.contrib.auth) e a renderização de formulários públicos (django.forms).

Como usar a Sandbox

Siga os passos abaixo para preparar o ambiente local de testes:

  1. 1. Instalar dependências e preparar o banco:
    Certifique-se de usar o gerenciador de pacotes uv para consistência: 
    uv run python manage.py migrate
  2. 2. Popular o banco (Seed):
    Popule permissões, grupos, dados simulados e usuários de testes: 
    uv run python manage.py seed_sandbox
  3. 3. Iniciar o servidor local:
    Suba o servidor de desenvolvimento: 
    uv run python manage.py runserver
  4. 4. Simular Sessões de Usuário:
    Acesse o painel QA Orchestrator para alternar instantaneamente entre perfis com diferentes níveis de permissão sem precisar digitar senhas.
